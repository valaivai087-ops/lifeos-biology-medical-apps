📚 核心統計概念 | 完整學習模板

平均數 (Mean) | 理解證據的第一步 ▼
📊 概念圖 (Concept Map)
患者群體的檢驗值 ↓ 計算所有值的總和 ↓ 除以人數 ↓ 得到平均值 ↓ 代表此群體的「典型」水平
🧠 心智圖 (Mind Map)
平均數 (Mean) ├─ 定義：所有值的總和÷個數 ├─ 公式：μ = Σx / n ├─ 優點：計算簡單、常用 ├─ 缺點：受極端值影響 ├─ 何時用：對稱分布時最代表 └─ 臨床應用：描述患者群體特徵
📐 公式整理
樣本平均數：M = Σx / n 母體平均數：μ = Σx / N 其中： Σx = 所有值的總和 n = 樣本數 N = 母體總數
💬 白話解釋

平均數就是把所有數字加起來再平均分配。想像班上30個學生的身高，把所有身高加起來除以30，就得到班上的「平均身高」。這個平均身高代表了班上「典型」學生的身高水平。但要注意，如果班上有一個特別高的學生（極端值），平均身高會被拉高。

🏥 臨床案例
案例：評估新型降血壓藥
某項臨床試驗給100位高血壓患者使用新藥A。

收縮壓變化（mmHg）：
患者1: -15, 患者2: -20, 患者3: -18, ... 患者100: -22

計算平均效果：
平均下降 = (-15 + -20 + -18 + ... + -22) / 100 = -18 mmHg

臨床意義：
新藥A平均能將患者收縮壓降低18 mmHg。醫生可以用這個平均值向患者解釋：「這個藥平均能降低你約18個單位的血壓」。
📄 Paper案例

論文： "Effects of Drug X on Blood Pressure in Hypertensive Patients" (2024)
報告方式： "The mean reduction in systolic BP was 18±4 mmHg (p=0.001)"
解讀： 平均下降18 mmHg，標準差4 (表示下降12-24 mmHg很常見)，P值0.001 (結果有統計顯著性)

⚠️ 常見誤解
❌ 錯誤
「平均血糖5 mmol/L，表示所有患者血糖都是5」
✅ 正確
「平均血糖5 mmol/L，表示患者群體的『典型』水平是5，但個別患者可能3-7」

10位患者用藥後血糖下降分別為：-5, -8, -6, -7, -9, -5, -10, -6, -8, -6 mmol/L。平均下降多少？

中位數 (Median) | 排除極端值的智慧 ▼
📊 概念圖
將所有患者的值排序 ↓ 找出中間位置的值 ↓ 該值就是中位數 ↓ 不受極端值影響
🧠 心智圖
中位數 (Median) ├─ 定義：排序後的中間值 ├─ 特點：不受極端值影響 ├─ 何時用：分布不對稱時 ├─ 優點：代表性更真實 ├─ 缺點：計算複雜度高 └─ 臨床應用：恢復時間、住院天數
📐 公式整理
奇數個數據：Median = 第(n+1)/2個值 偶數個數據：Median = [第n/2個 + 第(n/2+1)個] / 2 例如10個值排序後：3, 5, 7, 8, 9, 11, 12, 14, 16, 18 Median = (9 + 11) / 2 = 10
💬 白話解釋

中位數是「排隊後站在中間的人」。不管隊伍最前面有多矮的人，最後面有多高的人，中位數代表的永遠是中間位置。在醫學中，如果有少數特別好或特別差的患者，中位數比平均數更能代表「典型」情況。

🏥 臨床案例
案例：癌症患者生存時間
100位癌症患者追蹤生存時間（月）：
平均生存時間 = 25個月
中位生存時間 = 20個月

為什麼差異這麼大？
因為有少數患者活了很久（60-120個月），拉高了平均值。但「典型」患者活約20個月。中位數更準確反映患者群體的預後。
🎯 每日測驗

5位患者住院天數：2, 3, 4, 5, 45天。中位數和平均數分別是？

標準差 (Standard Deviation) | 衡量波動的尺子 ▼
📊 概念圖
計算每個值與平均值的距離 ↓ 平方這些距離 ↓ 求平均 ↓ 開平方 ↓ 得到標準差 ↓ 代表波動大小
🧠 心智圖
標準差 (SD) ├─ 定義：平均離散程度 ├─ 公式：σ = √[Σ(x-μ)²/n] ├─ 含義：SD越大，波動越大 ├─ 68-95-99.7規則 │ ├─ ±1σ: 68% │ ├─ ±2σ: 95% │ └─ ±3σ: 99.7% └─ 臨床應用：判斷數據分散度
📐 公式整理
樣本標準差：s = √[Σ(x-M)²/(n-1)] 母體標準差：σ = √[Σ(x-μ)²/n] 68-95-99.7 法則： 在常態分布中 • 68% 數據在 μ±1σ • 95% 數據在 μ±2σ • 99.7% 數據在 μ±3σ
💬 白話解釋

標準差就是「患者們到底有多不一樣」的尺子。SD越大，表示患者的檢驗值波動越大；SD越小，表示患者都很相似。例如兩間醫院的患者平均血糖都是100，但A醫院標準差是5（患者都在95-105），B醫院標準差是20（患者在80-120），B醫院的患者波動更大。

🏥 臨床案例
案例：血糖控制的評估
患者A：平均血糖 = 130±8 mmol/L
患者B：平均血糖 = 130±25 mmol/L

哪個患者血糖控制更好？
雖然平均值相同，但患者A的血糖波動小（SD=8），更穩定。患者B波動大（SD=25），時高時低，更容易出現低血糖風險。
🎯 每日測驗

在常態分布中，μ=100，σ=15。大約95%的數據應該落在哪個範圍？

敏感度 vs 特異度 | 診斷測試的雙重身份 ▼
📊 概念圖
所有患者 → 敏感度 = 多少患者被檢出 所有非患者 → 特異度 = 多少非患者被正確排除 敏感度 = 排除疾病（用於篩檢） 特異度 = 診斷疾病（用於確診）
🧠 心智圖
診斷測試 ├─ 敏感度 Se = TP/(TP+FN) │ ├─ 衡量：患者中有多少被檢出 │ ├─ 應用：用於排除疾病 │ └─ 篩檢需要高敏感度 ├─ 特異度 Sp = TN/(TN+FP) │ ├─ 衡量：非患者中有多少被排除 │ ├─ 應用：用於確認疾病 │ └─ 確診需要高特異度 └─ ⚠️ 不能單獨評估測試好壞
📐 公式整理
敏感度 (Sensitivity) = TP / (TP + FN) 特異度 (Specificity) = TN / (TN + FP) 其中： TP = True Positive (真陽性，患者，測試陽性) FN = False Negative (假陰性，患者，測試陰性) TN = True Negative (真陰性，非患者，測試陰性) FP = False Positive (假陽性，非患者，測試陽性)
💬 白話解釋

敏感度：「測試在患者中的找人能力」。敏感度95%表示100個患者中有95個被檢出，5個被漏掉。適合篩檢，因為不能漏掉患者。

特異度：「測試在非患者中的排除能力」。特異度95%表示100個非患者中有95個被正確排除，5個被誤診為患者。適合確診，因為不能誤診健康人。

🏥 臨床案例
案例：新冠檢測的應用
核酸檢測：Se=95%, Sp=99%
快速抗原檢測：Se=80%, Sp=99%

篩檢階段（大規模檢測）：
用核酸檢測（高敏感度），找出所有感染者，漏檢風險小。

確診階段（症狀患者）：
兩者都可，但核酸更可靠，確認患者確實感染。

某診斷測試用於篩檢常見疾病，下列參數組合最適合？

P值 (P-value) | 最常被誤解的統計概念 ▼
📊 概念圖
虛無假設：沒有效果 ↓ 計算觀察到現有數據的機率 ↓ 假如虛無假設真的成立 ↓ 就能得到P值 ↓ P值 ≠ 虛無假設成立的機率
💬 最常見的5大誤解
❌ 誤解1： P值是虛無假設為真的機率
✅ 正確： P值是假設虛無假設為真，看到這個數據的機率

❌ 誤解2： P < 0.05代表發現很重要
✅ 正確： P < 0.05只表示統計顯著，不等於臨床重要

❌ 誤解3： P值是結果真實的機率
✅ 正確： P值不能告訴你結果是否真實

❌ 誤解4： P > 0.05表示沒有效果
✅ 正確： P > 0.05表示證據不足，可能有效果但沒檢出

❌ 誤解5： 多次檢測P值，選擇小於0.05的結果
✅ 正確： 這是P-hacking，會製造假發現
📐 公式整理
P值 = 在虛無假設下，觀察到現有數據或更極端結果的機率 常見的α (顯著性水平)： • α = 0.05 (5%) - 最常用 • α = 0.01 (1%) - 更嚴格 • α = 0.001 (0.1%) - 極為嚴格 判斷： • P < α → 拒絕虛無假設（有統計顯著性） • P ≥ α → 不能拒絕虛無假設（無足夠證據）
🏥 臨床案例
案例：新藥降血壓的P值解讀
研究報告：「新藥A相比標準治療，血壓下降更多，P = 0.03」

❌ 錯誤理解：
「新藥真的更好的機率是97%」

✅ 正確理解：
「假如新藥根本沒有作用，我們觀察到這個結果（或更極端結果）的機率是3%」

臨床決策：
P < 0.05有統計顯著性，但還要看：效果有多大？有沒有副作用？費用高不高？
🎯 每日測驗

一項研究P值=0.06，下列說法何者最正確？

信賴區間 (Confidence Interval) | 比P值更有信息量 ▼
📊 概念圖
計算點估計 ↓ 估計標準誤 ↓ 加上和減去邊界 ↓ 形成一個區間 ↓ 相信真實參數在其中
📐 公式整理
95% CI = 點估計 ± 1.96 × SE 例如： 平均血壓下降 = 15 mmHg 95% CI: 10-20 mmHg 解釋： 真實的血壓下降值有95%的信心落在10-20 mmHg之間 與P值的關係： 如果95% CI 不包含0，則P < 0.05
💬 白話解釋

信賴區間就是「如果我們重複做這個研究100次，約95次的區間會包含真實值」。比P值更好用，因為它同時告訴你「估計值是多少」和「估計的精確度如何」。區間越窄，估計越精確。

🏥 臨床案例
案例：兩個不同的研究結果
研究A：新藥降血壓 15 mmHg, 95% CI: 14-16 mmHg
研究B：新藥降血壓 15 mmHg, 95% CI: 5-25 mmHg

哪個研究更可信？
研究A的CI很窄（14-16），估計精確，結果可信。
研究B的CI很寬（5-25），估計不精確，實際效果可能很小也可能很大。
兩者平均值相同，但A明顯更好。
🎯 每日測驗

如果95% CI包含0（例如-2到+5），代表什麼？

相對風險 (Relative Risk) | 風險因子的力量 ▼
📐 公式整理
RR = 風險(暴露組) / 風險(非暴露組) = [a/(a+b)] / [c/(c+d)] 解釋： • RR = 1：沒有關聯 • RR > 1：暴露增加風險（危害因子） • RR < 1：暴露減少風險（保護因子） • RR = 2：暴露組風險是非暴露組的2倍
💬 臨床解釋

RR是最直觀的風險指標。例如吸煙者患肺癌的相對風險是15，表示吸煙者患肺癌的風險是非吸煙者的15倍。用於隊列研究或RCT，能直接比較風險。

🏥 臨床案例
案例：高血壓用藥和中風風險
研究結果：
用降壓藥組：發生中風的風險 = 5%
未用藥組：發生中風的風險 = 15%

RR = 5% / 15% = 0.33

解釋：
用藥組的中風風險是未用藥組的33%，或者說降壓藥將中風風險降低了67%。
勝算比 (Odds Ratio) | 病例對照研究的語言 ▼
📐 公式整理
OR = (a × d) / (b × c) 其中： a = 患者+暴露 b = 患者-暴露 c = 非患者+暴露 d = 非患者-暴露 解釋： • OR = 1：沒有關聯 • OR > 1：暴露與疾病正相關 • OR < 1：暴露與疾病負相關 • 在罕見疾病中，OR ≈ RR
💬 何時使用OR

病例對照研究：必須用OR，因為無法計算風險（患者數量固定）。
隊列研究：用RR更合適。
罕見疾病：OR近似等於RR，可以簡化計算。

🏥 臨床案例
案例：特定基因與乳癌風險
病例對照研究（100位乳癌患者 vs 100位健康女性）：

有BRCA突變：患者80人，健康人10人
無BRCA突變：患者20人，健康人90人

OR = (80×90) / (20×10) = 36

解釋：
有BRCA突變的女性罹患乳癌的勝算是無突變女性的36倍。
常態分布 (Normal Distribution) | 自然界最常見的形狀 ▼
🧠 心智圖
常態分布 ├─ 形狀：鐘形曲線 ├─ 特性： │ ├─ 平均 = 中位數 = 眾數 │ ├─ 對稱（左右一致） │ └─ 永不接觸X軸 ├─ 68-95-99.7 規則 └─ 為什麼重要： ├─ 許多生物醫學變量 ├─ 許多統計測試的基礎 └─ 樣本均值也常態分布
📐 公式整理
常態分布概率函數： f(x) = (1/(σ√(2π))) × e^(-(x-μ)²/(2σ²)) 標準化（Z-score）： z = (x - μ) / σ Z-score 的意義： • z = 0：值等於平均 • z = 1：值高於平均1個標準差 • z = -1：值低於平均1個標準差 • z > 3 或 z < -3：極端值（罕見）
💬 白話解釋

常態分布就是「最自然的分布方式」。大部分人聚集在中間，兩端逐漸減少，形成鐘形。身高、智力、血糖等很多生物醫學指標都近似常態分布。許多統計測試（t檢驗、方差分析）都假設數據常態分布。

🏥 臨床案例
案例：實驗室參考範圍的設定
100位健康人的血糖測量：
平均值 = 100 mg/dL
標準差 = 10 mg/dL

假設常態分布，參考範圍（95%）：
= 100 ± 1.96 × 10
= 80.4 - 119.6 mg/dL

解釋：
95%的健康人血糖在80-120範圍內。超出此範圍可能異常。
眾數 (Mode) | 最常見的值 ▼
📊 概念圖
計算每個值出現的次數 ↓ 找到出現最多的值 ↓ 那就是眾數 ↓ 代表最「受歡迎」的值
🧠 心智圖
眾數 (Mode) ├─ 定義：出現次數最多的值 ├─ 特點：可以有多個眾數 ├─ 單峰/雙峰/多峰分布 ├─ 何時用：分類數據 ├─ 優點：直觀、易理解 ├─ 缺點：不精確、不穩定 └─ 臨床應用：最常見症狀、最常見基因型
💬 白話解釋

眾數就是「最流行的」。在一間醫院，如果患者的最常見診斷是「2型糖尿病」，那「2型糖尿病」就是眾數。不用計算，只要看什麼最多就是眾數。特別適合用於描述分類數據。

🏥 臨床案例
案例：急診病人診斷統計
某家醫院今天100位急診患者診斷：
腹痛：35人
頭痛：28人
胸痛：20人
其他：17人

眾數：腹痛

這提示醫院應該加強腹痛相關的診療能力，因為這是最常見的問題。
🎯 每日測驗

某科室一周內患者主訴：咳嗽12人、發熱15人、腹瀉8人、咳嗽12人。眾數是？

抽樣 (Sampling) | 從部分推斷整體 ▼
📐 公式整理
標準誤 (SE) = σ / √n 其中：σ是母體標準差，n是樣本量 樣本量計算： n = (z²σ²) / E² 其中：z是信度值，E是允許誤差
💬 核心概念

隨機抽樣：每個個體有相等被選中機率，最能代表母體，消除選擇偏誤。
分層抽樣：先分層，再層內隨機抽樣，確保各層都有代表。
系統抽樣：按固定間隔選擇，簡單但可能有偏誤。
便利抽樣：選最容易的對象，偏誤最大，應避免用於科研。

⚠️ 常見偏誤
選擇偏誤：樣本不代表母體
無應答偏誤：拒絕參加的人有特異性
測量偏誤：測量方法有系統性錯誤
機率 (Probability) | 事件發生的可能性 ▼
📐 公式整理
基本機率：P(A) = 有利結果數 / 所有可能結果數 條件機率：P(A|B) = P(A∩B) / P(B) 全機率公式：P(A) = Σ P(A|Bᵢ) × P(Bᵢ) 貝氏定理：P(A|B) = P(B|A) × P(A) / P(B)
💬 臨床應用

診斷的本質是用機率決策。病人檢測陽性，他真的有病的機率是多少？這涉及測試敏感度、特異度和患病率的組合——正是貝氏定理的應用。

假說檢驗 (Hypothesis Testing) | 科學推論的框架 ▼
📊 流程圖
提出問題 ↓ 建立虛無假設 (H₀) 和對立假設 (H₁) ↓ 選擇適當的統計測試 ↓ 計算測試統計量和P值 ↓ 與α水平比較 ↓ 得出結論
📐 公式整理
第一類錯誤 (α)：拒絕真實的H₀ (假陽性) 第二類錯誤 (β)：接受虛假的H₀ (假陰性) 統計檢力 = 1 - β (正確拒絕虛假H₀的機率) 常見α：0.05, 0.01, 0.001 常見β：0.20 (80%檢力)
效應量 (Effect Size) | 不只統計，更要臨床 ▼
📐 公式整理
Cohen's d = (M₁ - M₂) / SD 解釋：d = 0.2 (小), 0.5 (中), 0.8 (大) 相對風險 (RR) = Risk(E+) / Risk(E-) 勝算比 (OR) = (a×d) / (b×c) 數字需要背景才有意義： P < 0.05 ✓ 統計顯著 d = 0.8 ✓ 效應大 但若治療成本是收益的100倍，就不值得
💬 關鍵區別

統計顯著 ≠ 臨床重要
大樣本下，即使極小的差異也能達到P < 0.05。但如果效應量很小（d = 0.1），臨床上可能無關緊要。反之，小樣本可能無法檢出真實的重要效應。

ROC曲線 (ROC Curve) | 診斷測試的全景圖 ▼
📐 公式整理
ROC曲線：以1-特異度為X軸，敏感度為Y軸 AUC (Area Under Curve)： • 0.5：無診斷價值 • 0.6-0.7：差 • 0.7-0.8：中等 • 0.8-0.9：很好 • > 0.9：優秀 • 1.0：完美測試 AUC = 0.8表示：隨機選一個患者和非患者， 該測試有80%的機率正確判斷誰是患者
💬 臨床應用

ROC曲線幫助醫生在不同敏感度-特異度水平中選擇最適合的臨界值。篩檢（寧可多檢一些）時用敏感度優先的臨界值；確診（不想誤診）時用特異度優先的臨界值。

統合分析 (Meta-analysis) | 整合多項研究 ▼
📊 概念圖
系統性搜索相關文獻 ↓ 制定納入排除標準 ↓ 評估各項研究質量 ↓ 提取數據 ↓ 統計整合分析 ↓ 評估發表偏誤和異質性 ↓ 得出綜合結論
💬 優勢和限制
✅ 優勢：
• 樣本量大，統計檢力高
• 結論更穩健
• 1級最高級別證據

❌ 限制：
• 垃圾進垃圾出 (GIGO)
• 發表偏誤（只發表陽性結果）
• 異質性可能掩蓋真相
系統性回顧 (Systematic Review) | 最高級別的證據 ▼
🧠 心智圖
系統性回顧 ├─ 定義：系統、明確、可重複的文獻回顧 ├─ 通常配合統合分析 ├─ 1級最高級別證據 ├─ 與非系統性回顧的區別： │ ├─ 明確的搜索策略 │ ├─ 預先註冊（PROSPERO） │ ├─ 獨立評審員評估 │ └─ 可重現的方法 └─ 臨床應用：指南制定、最佳實踐證據
💬 如何評讀SR

✓ 是否在PROSPERO預先註冊？
✓ 納入的研究都是高質量嗎？
✓ 有多少研究支持結論？
✓ 是否有利益衝突？
✓ 結論是否過度解釋？

🔬 證據醫學等級評估 | Evidence Hierarchy

請選擇該研究的等級：

等級 1a：系統評論 (SR) 的多個RCT薈萃分析
最高質量證據。多項隨機試驗的系統性整合分析。
等級 1b：高質量RCT或SR的單一大型隨機試驗
設計嚴謹的隨機對照試驗，樣本量大，隨訪完整。
等級 1c：全或無型臨床試驗結果
治療效果極端明顯，無需對照組。
等級 2a：SR的優質隊列研究
多項質量較好的隊列研究的系統性評論。
等級 2b：低質量RCT或單一隊列研究
前瞻性隊列研究，但設計有限制或樣本量較小。
等級 3a：SR的病例對照研究
多項病例對照研究的系統評論。
等級 3b：單一病例對照研究
後溯性設計，易受偏誤影響。
等級 4：病例系列報告
沒有對照組的描述性研究。
等級 5：專家意見
最低質量證據。基於臨床經驗或傳統。
✅ 你的評估

🎯 GRADE 品質評估方法

GRADE (Grading of Recommendations, Assessment, Development, and Evaluations) 評估證據品質的四個維度：

1️⃣ 研究設計質量

RCT起點高，觀察研究起點低

2️⃣ 偏誤風險

設計缺陷、實施不佳、報告不完整

3️⃣ 一致性

不同研究結果是否一致

4️⃣ 直接性

研究對象是否代表臨床患者

🏥 臨床案例庫 | 證據應用實戰

案例1：降血壓藥的選擇困境

臨床情景：
55歲男性患者，高血壓5年，BP 150/95。醫生考慮給予ACE抑制劑或鈣離子通道阻滯劑。患者詢問：「哪個藥更好？」

相關證據：
• 2024年高血壓指南（1級證據：多個RCT薈萃分析）：兩類藥物同樣有效
• 該患者沒有蛋白尿，無需特別選擇ACE抑制劑
• 但患者有輕度肺功能下降，鈣離子通道阻滯劑可能更安全

EBM決策：
用強有力的證據（1級）說明兩者同樣有效，但根據患者個人情況（肺功能）選擇鈣離子通道阻滯劑，並向患者解釋理由。

案例2：新藥的臨床適用性問題

臨床情景：
某新型抗癌藥在期刊發表，相比舊藥延長生存期3個月。患者看到新聞，要求使用新藥。

批判性評讀：
✓ 這是2級研究嗎？（隊列研究）
✓ 樣本量多少？
✓ 隨訪周期夠嗎？
✓ 副作用如何？生活質量如何？
✓ 新藥花費多少倍？值不值得？

EBM決策：
用證據分析回答患者：「延長3個月生存是統計顯著，但臨床上意義有限。考慮副作用、費用、生活質量，我們需要共同決策。」

案例3：信賴區間的臨床應用

臨床情景：
兩家醫院分別進行新穎治療效果研究，都報告「平均效果：降低肝酶20%」

詳細數據：
A醫院：下降20%, 95% CI: 18-22%（樣本1000人，隨訪完善）
B醫院：下降20%, 95% CI: 5-35%（樣本50人，隨訪不完全）

EBM判斷：
雖然平均值相同，但A醫院的CI很窄，說明結果精確可靠，有信心效果確實在18-22%。
B醫院的CI很寬，說明結果不精確，效果可能小到5%，也可能大到35%，無法確定。
應該選擇相信A醫院的結果。

案例4：敏感度特異度的臨床權衡

臨床困境：
診斷罕見疾病時，有兩個測試可選：

快速測試： 敏感度95%，特異度70%
精確測試： 敏感度90%，特異度98%

EBM決策：
篩檢階段：用快速測試，因為我們不能漏診患者
確診階段：用精確測試，因為我們不能誤診
結合應用：先用快速測試篩檢，陽性患者再用精確測試確診，既提高效率又保證準確

案例5：P值誤解導致的臨床錯誤

案例：
某研究發現新保健食品能「預防感冒」，P = 0.04

❌ 錯誤理解：
「有96%的機率這個食品真的能預防感冒」
「食品預防感冒的效果非常顯著」
「我應該推薦所有患者吃」

✅ 正確理解：
「假如食品根本無效，看到這個結果的機率是4%」
「結果有統計顯著性，但效果大小可能很小」
「需要看：樣本量多少？效果大小多少？有無副作用？」

EBM決策：
P < 0.05只是門檻，不代表結果重要。需要進一步檢查：
✓ 信賴區間是什麼？
✓ 樣本量夠嗎？
✓ 有沒有多次比較問題（P-hacking）？
✓ 實際效果值得臨床應用嗎？
